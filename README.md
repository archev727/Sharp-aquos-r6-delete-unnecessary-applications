# Sharp-aquos-r6-delete-unnecessary-applications



IN bat file contains commands ADB allowing you to remove unnecessary applications like softbank and other crap


Usage: 

1) Download the archive and unzip the files to a folder on the C drive.

2) In the folder path line, click on the path, and then write cmd

3) Enter the command "adb devices" and confirm the USB debugging on the phone (don't forget to enable it in the developer settings)

4) After finding the smartphone in adb, drag the file "CLEAN_R6" into the command line, and then press enter

5) Wait until the program finishes working and then use it!
